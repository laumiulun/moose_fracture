# GradientComponent
!syntax description /Kernels/GradientComponent

Implements

$$
u - \nabla_\alpha v,
$$

where $u$ is the kernel variable, $\alpha$ (`component`) is a coordinate system
direction, and $v$ is a coupled variable.
